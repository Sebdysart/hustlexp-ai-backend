# Backend scripts

Scripts here are **backend-specific** only. Use repo-root **`scripts/`** for shared tooling (migration safety, schema diff, consolidate migrations, validate schema, incident report, etc.).

- `classify-pr-changes.ts` — PR change classification
- `compute-readiness-score.ts` — Readiness score
- `concurrency-load-test.ts` — Concurrency load test
- `revenue-replay.ts` — Revenue replay
